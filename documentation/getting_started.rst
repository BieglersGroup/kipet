Getting started
======================================

To use KIPET, import the package from within a python console::

	import kipet	

* Import KIPET
* Build a template model 
* Simulate model
* Estimate variances
* Estimate parameters
* Compute confidence intervals
* Plot concentration and absorbance profiles
